# Amazon Sales & Revenue Dashboard

This project is a Power BI dashboard built to analyze Amazon sales data.

## Features:
- KPI tracking (Revenue, Profit, Orders, Quantity)
- Sales trend analysis
- Category and product performance
- State-wise revenue insights
- Interactive filters

## Tools Used:
- Power BI

## Insights:
- Revenue peaked in 2023
- Electronics & Clothing dominate sales
- Top products contribute majority revenue
- Texas generates highest revenue
